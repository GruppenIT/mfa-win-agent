Development, Maintenance and Support
====================================

The GruppenMFA Credential Provider was first developed by Last Squirrel IT [#lastsquirrel]_.
The company has a long experience in Microsoft Windows security tools. They developed many different credential providers and
plugins for Active Directory Federation Services.
 
Since 2018 the development is continued by Gruppen it Security.
You will get maintenance and support via the company Gruppen it Security [#gruppenit]_.
Gruppen it Security also maintains the GruppenMFA Authentication System and issues
different service level agreements [#sla]_ for the GruppenMFA Credential
Provider, the GruppenMFA Authentication System and many other plugins.
