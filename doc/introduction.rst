Introduction
============

The GruppenMFA Credential Provider is a tool to improve the logon security
of your Windows Desktops, Servers and Windows Terminal Servers.
It is used to add a second factor for authentication, when logging into your
Windows system.

The GruppenMFA Credential Provider does this by communicating with the GruppenMFA Authentication System [#gruppenmfa]_.
The GruppenMFA Authentication System can manage many different kinds of second factors for the domain users: Passkeys/WebAuthn token, 
PUSH token, Classic OTP token like HOTP, TOTP, Email or SMS, Yubikeys and many more!

Additonally, GruppenMFA can provide flexible MFA workflows with its powerful policy and event system. The GruppenMFA Credential Provider
also offers a wide range of configuration options to customzie the login to your needs.

.. [#gruppenmfa] https://gruppenmfa.org
